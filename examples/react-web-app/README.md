# React Web Example App

Example React web application demonstrating endura's durable workflow execution.

## Coming Soon

This example app is under development. It will showcase:

- Setting up endura with in-memory storage for web
- Defining and registering workflows
- Starting workflow executions
- Using React hooks for UI integration
- Handling browser refresh and persistence

## Prerequisites

- Node.js 20+

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
