# Expo Example App

Example Expo/React Native application demonstrating endura's durable workflow execution.

## Coming Soon

This example app is under development. It will showcase:

- Setting up endura with expo-sqlite
- Defining and registering workflows
- Starting workflow executions
- Using React hooks for UI integration
- Background task processing
- Handling offline scenarios and app restarts

## Prerequisites

- Node.js 20+
- Expo CLI
- iOS Simulator or Android Emulator

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npx expo start
```
